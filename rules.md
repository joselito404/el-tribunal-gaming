# Información del Proyecto: El Tribunal Gaming

## Objetivo del Agente
- Tu función principal es **mantener** y **ejecutar** las directivas de código enviadas por el usuario. 
- No debes proponer cambios arquitectónicos por tu cuenta, solo actuar sobre lo que se te pide.

## Contexto Social
- El proyecto es para un grupo de 6 amigos: Jose (Director), Mario, Iván, Carmen, Lázaro y Alejandro.

## Guía de Estilo y Estructura (Mantener siempre)
- **Tecnologías:** React (Vite), Tailwind CSS y Firebase Firestore.
- **Estética:** Siempre mantener el estilo Dark/Neón (paleta de Indigo, Pink y Slate).
- **Componentes:** Mantener la estructura de componentes funcionales y el uso de la librería 'lucide-react' para iconos.
- **Responsividad:** Asegurarse de que cualquier pequeño cambio de UI mantenga la compatibilidad con móviles y escritorio.

## Notas de Operación
- El usuario suele pedir tareas pequeñas y puntuales (ej. corregir una imagen o un enlace).
- **NO** es necesario que actualices el Changelog o la Bitácora a menos que se te pida explícitamente; esa parte se gestiona de forma externa.
- Si ves una imagen rota o un error de importación al realizar una tarea, corrígelo silenciosamente para mantener la estabilidad.
